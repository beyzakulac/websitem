#  Kişisel Web Projem

Merhaba! Bu proje, web geliştirme temellerini pekiştirmek ve kullanıcı deneyimine dayalı web uygulamaları geliştirmek amacıyla hazırladığım kişisel portfolyo çalışmamdır.

##  Kullanılan Teknolojiler
Projeyi modern web standartlarına uygun şekilde geliştirdim:
- **HTML5:** Anlamsal (semantic) etiketler kullanılarak oluşturulan sağlam bir iskelet.
- **CSS3:** Flexbox ve modern yerleşim teknikleri ile responsive tasarım.
- **JavaScript:** Uygulamanın işlevselliği ve etkileşimli özellikler için doğrudan HTML dosyaları içerisinde yer alan betikler.

##  Temel Özellikler
- **Responsive Tasarım:** Mobil ve masaüstü cihazlarla tam uyumlu görünüm.
- **Dinamik İçerik:** HTML dosyaları içerisine gömülü JavaScript kodları sayesinde kullanıcı etkileşimi sağlayan özellikler.
- **Veri Kalıcılığı:** Tarayıcı tabanlı depolama yöntemleri ile kullanıcı verilerini yönetme kapasitesi.

##  Proje Yapısı
- `index.html`: Ana sayfa yapısı ve etkileşimli JavaScript mantığı.
- `contact.html`: İletişim sayfası ve form yönetimi.
- `style.css` / `resume.css`: Görsel tasarımlar ve düzenleme.
- `*.css` dosyaları: Sayfalara özel stil tanımlamaları.
